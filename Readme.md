# 🌆 Panduan Pemasangan Cache Kota - District Khatulistiwa Roleplay

Guna meningkatkan kenyamanan bermain dan mempercepat proses *loading screen* saat memasuki kota, kami menyarankan seluruh warga untuk memasang **Cache Kota** secara manual. 

Dengan memasang cache ini, PC Anda tidak perlu mengunduh ulang aset-aset berat (seperti kendaran, baju, MLO/bangunan) langsung dari server saat *connecting*.

---

## 📥 Syarat & Persiapan
1. Pastikan Anda sudah mengunduh file `cache-kota.zip` melalui tombol resmi di website kami.
2. Pastikan aplikasi FiveM Anda dalam keadaan **tertutup (tidak sedang berjalan)**.
3. Siapkan aplikasi pengekstrak file seperti **WinRAR** atau **7-Zip**.

---

## 🛠️ Tutorial Langkah demi Langkah

### Langkah 1: Buka Folder FiveM Application Data
Anda harus masuk ke direktori tempat FiveM menyimpan data *cache*. Cara termudahnya:
1. Klik kanan pada icon **FiveM** di Desktop atau Start Menu Anda.
2. Pilih **Open file location**.
3. Cari dan masuk ke folder yang bernama 📂 **FiveM Application Data** (biasanya ditandai dengan icon berlambang siput).

### Langkah 2: Masuk ke Folder Cache Utama
Di dalam folder *FiveM Application Data*, Anda akan melihat beberapa folder sistem.
1. Cari dan buka folder bernama 📂 **data**.
2. Di dalam folder `data`, Anda akan menemukan folder bernama 📂 **server-cache-priv** atau 📂 **server-cache**.

### Langkah 3: Ekstrak File Cache Kota
1. Buka file `cache-kota.zip` yang sudah Anda download sebelumnya.
2. **Ekstrak/Copy** seluruh isi folder atau file `.bin` yang ada di dalam zip tersebut.
3. **Paste (Tempel)** ke dalam folder 📂 **server-cache-priv** yang Anda buka di Langkah 2.
4. Jika muncul notifikasi konfirmasi, pilih **"Replace the files in the destination"** (Ganti file yang sudah ada).

### Langkah 4: Selesai & Mainkan!
Sekarang folder cache Anda sudah terisi dengan aset-aset Kota District Khatulistiwa. 
* Silakan buka kembali FiveM Anda.
* Klik tombol **Mulai Bermain** di website atau langsung *connect* ke server.
* Rasakan proses *loading masuk kota* yang jauh lebih cepat!

## ❓ Pertanyaan yang Sering Diajukan (FAQ)

> **Q: Apakah aman memasang cache secara manual?**
> **A:** Sangat aman. Ini adalah aset standar server (seperti mobil, baju, dan bangunan) yang dibundel menjadi satu agar Anda tidak perlu *download* satu per satu di dalam game.

> **Q: Kenapa setelah pasang cache, loading pertamanya masih agak tersendat?**
> **A:** Pada *login* pertama setelah pemasangan, FiveM akan melakukan validasi/sinkronisasi index aset. Ini hanya terjadi sekali, untuk login berikutnya akan langsung instan.

> **Q: Kapan saya harus download ulang cache kota ini?**
> **A:** Kami menyarankan untuk memperbarui cache ini jika server baru saja melakukan *update* besar-besaran (misalnya pergantian musim atau penambahan fitur besar). Pengumuman akan selalu diberikan di Discord.
